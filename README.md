# ShadowOS

一个基于浏览器的模拟操作系统，提供完整的桌面体验。

## 特性

### 桌面系统
- 赛博朋克风格全屏渐变背景
- 可拖拽的桌面图标（支持自由定位）
- 右键菜单（新建终端、文件管理器、浏览器、锁屏等）
- 锁屏界面（显示时间、日期）

### 窗口管理
- 窗口拖动（边界限制）
- 窗口层级管理（点击置顶）
- 窗口最小化/最大化/关闭
- 窗口打开/关闭动画
- 多窗口同时运行

### 系统功能
- **任务栏**：开始菜单、搜索、系统托盘（通知/锁屏）、实时时钟
- **全局搜索**：`Ctrl+K` 快速搜索应用和文件
- **通知中心**：系统操作通知，自动消失或手动清除
- **快捷键**：
  - `Ctrl+K` 全局搜索
  - `Ctrl+W` 关闭当前窗口
  - `Ctrl+T` 打开终端
  - `Ctrl+N` 打开记事本
  - `Ctrl+C` 打开计算器
  - `Esc` 关闭搜索
- **天气小组件**：桌面实时天气展示，支持多城市切换
- **番茄钟小组件**：桌面专注计时器，支持专注/休息模式切换
- **右键菜单**：桌面右键快速操作
- **主题系统**：青色/紫色/绿色色调切换
- **壁纸系统**：6 种预设壁纸 + 自定义上传壁纸

## 技术栈

- **HTML5** — 页面结构
- **CSS3** — 赛博朋克风格样式（毛玻璃、动画、渐变）
- **原生 JavaScript** — 无框架，零依赖（除 CodeMirror 语法高亮库）
- **IndexedDB** — 文件持久化存储
- **LocalStorage** — 主题、壁纸、已安装插件等配置持久化
- **CodeMirror** — 代码编辑器语法高亮
- **Open-Meteo API** — 实时天气数据（免费、无需注册）

## 快速开始

```bash
# 方式 1：使用 Node.js
npx serve

# 方式 2：使用 Python
python -m http.server 8080

# 方式 3：直接打开 index.html 文件
```

然后在浏览器中访问 `http://localhost:3000` 即可。
